What the app does

🔬 Compute descriptors

Type or paste a SMILES string and get MW, logP, TPSA, QED, Lipinski, HOMO/LUMO — instantly via the cloud API.

💊 Drug-likeness screen

Run Lipinski, Veber, and PAINS filters. See which rules pass or fail at a glance.

🔎 Similarity search

Find structurally similar molecules in the Q-Mol dataset using ECFP4 fingerprints.

📊 Usage dashboard

See your API quota, monthly usage by endpoint, and referral earnings — from anywhere.

🧪 Batch upload

Upload a CSV of SMILES from your phone's storage and queue a batch compute job.

📥 Offline results

Export your last compute results to CSV for offline use and sharing.

How to install

  1. Download the APK

    Tap the "Download Debug APK" button above. Your browser will save the file.

  2. Allow unknown sources

    Go to Settings → Apps → Special app access → Install unknown apps and allow your browser or file manager.

  3. Tap the downloaded file

    Open your downloads folder and tap qmol-android.apk to begin installation.

  4. Sign in with your API key

    Use the API key you received when you signed up or subscribed.

Requirements

  • Android 7.0 (API 24) or newer
  • Internet connection (computation happens in the cloud)
  • A free or paid Q-Mol API key
  • ≈ 6 MB storage for the app

For developers

The Android app is open source and lives in the android/ folder of the GitHub repository. It is a native WebView wrapper around the Q-Mol web app. Pull requests welcome.

git clone https://github.com/bobbybibi/Qmol-3.git
cd Qmol-3/android
./gradlew assembleDebug
