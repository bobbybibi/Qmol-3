Developer-first.

curl it. pip it. Or drop the OpenAPI spec into Postman. 47 endpoints, one authenticated header.

# compute descriptors for 2 molecules
curl -X POST https://api.qmol.dev/compute \
  -H "x-api-key: $QMOL_KEY" \
  -H "content-type: application/json" \
  -d '{"smiles":["CCO","c1ccccc1"]}'

# same thing from Python
from qmol_client import Client
c = Client(api_key="qk_…")
result = c.compute(["CCO", "c1ccccc1"])
for r in result:
    print(r["smiles"], r["mw"], r["logp"])

# bulk upload an SDF, get back Parquet
curl -X POST https://api.qmol.dev/upload/compute \
  -H "x-api-key: $QMOL_KEY" \
  -F "file=@library.sdf"

# retrosynthesis — one-step disconnection
curl -X POST https://api.qmol.dev/retro \
  -H "x-api-key: $QMOL_KEY" \
  -d '{"smiles":"CC(=O)Nc1ccccc1"}'
# => amide_coupling: CC(=O)O + Nc1ccccc1
